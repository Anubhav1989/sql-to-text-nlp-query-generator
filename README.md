# SQL to Text – Natural Language Query Generator

## Overview
An NLP-powered application that converts plain-English questions into 
executable SQL queries, enabling non-technical users to query databases 
without any SQL knowledge.

## Tech Stack
Python | Streamlit | Seq2Seq Model | NLP | SQL

## Features
- Converts natural language input into accurate SQL queries automatically
- Interactive Streamlit UI for non-technical users
- Fine-tuned sequence-to-sequence model for high query accuracy
- Supports diverse database schemas and query types

## How to Run
pip install -r requirements.txt
streamlit run app.py
